# EVIDEX Multi-Chain Evidence Platform

🚀 **Universal Multi-Chain Evidence Anchoring Platform (Enterprise Edition)**

A revolutionary blockchain-based evidence anchoring platform that enables users to upload digital files and permanently anchor their cryptographic proofs across 80+ blockchain networks. Optimized for enterprise trust, audit logs, and legal discovery.

## ✨ **Key Platform Upgrades**

- **💸 Merkle Tree Cryptographic Compression**: Native batching engine hashes thousands of files into a Single Master Root Hash, slashing blockchain gas fees by 99% while maintaining mathematical zero-knowledge proofs.
- **🤖 A.I. Smart Block Routing**: Algorithmically selects the cheapest and fastest layer-2 network (Polygon vs Arbitrum) dynamically during high congestion.
- **🛡️ 100% Uptime Fallback Mechanics**: Automatic exponential backoff and absolute cross-chain mitigation. If the primary ledger goes offline, Evidex seamlessly anchors to a secondary ledger to guarantee immutability.
- **✅ Audit Dashboard & Proof Certificates**: Completely overhauled Verification UI that securely generates W3C-compliant Verifiable JSON Credentials and verifiable cryptographic receipts for third-party auditors.

## 🛠️ **Technology Stack**

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS (Dark Trust Theme)
- **Backend**: Node.js, Express.js, Prisma ORM
- **Database**: PostgreSQL with Row-Level Security (RLS) Multi-Tenant Privacy
- **Blockchain**: Ethers.js, Polkadot.js, Bitcoin.js, Merkle Tree Cryptography
- **Infrastructure**: Docker, Turborepo, Vercel

## 🎯 **Use Cases**

- **⚖️ Legal**: Tamper-proof evidence for court submissions and decentralized discovery.
- **💰 Financial**: Immutable financial logging and cross-chain audit trails.
- **📰 Journalism**: Instant proof-of-existence anchoring for leaked documentation or reporting.
- **🏢 Compliance**: Contract timestamping and intellectual property protection.

## 🚀 **Quick Start**

```bash
# Clone the repository
git clone https://github.com/KumarAditya1729/Evidex.git
cd Evidex

# Install dependencies using pnpm
pnpm install

# Build the Monorepo
pnpm run build

# Start the Web Application
cd apps/web && pnpm run dev
```
